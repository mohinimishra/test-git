# Learning Git

## Common Commands

* Clone a repository
```bash
git clone <repo url>
```

* Check the Status
```bash
git status
```

* Add Changes to stage
```bash
# To add all the changes
git add .

# To add a particular file
git add <file name>
```

* To Commit the Files with Message
```bash
git commit -m "message"
```

* To Push the Files
```bash
git push origin <branch name>
```
